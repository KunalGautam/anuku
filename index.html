Anuku

Content Management Simplified

Anuku is Content Management System Simplified. It just works "out of the box". You can use it to setup a simple website with zero or minimal configuration.

Anuku needs "zero configuration" to start with and is great in design. It looks/behaves very elegantly on all browsers/devices (we still recommend not to fall back to IE8 or below for obvious reasons).

View screenshots of Anuku

Using:

  1. Php as the coding language

  2. MySQL as database

  3. Git for version control

  4. Bootstrap for UI
